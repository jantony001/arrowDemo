Node server requires the following environment variables:
	OPENSHIFT_MYSQL_DB_HOST
	
	OPENSHIFT_MYSQL_DB_PORT
	
	OPENSHIFT_MYSQL_DB_USERNAME
	
	OPENSHIFT_MYSQL_DB_PASSWORD
